# 启动

```bash
node server.js
```

# 浏览

打开浏览器，访问 http://localhost:3000/ 地址进行游戏交互操作

# 目录说明

```
├── code
|  ├── directiveLib.js          操作指令集
|  └── index.js                 核心代码
├── v1
|  └── index.js                 ”猜拳游戏“ 命令行版本
└── v2
   ├── action.js                ”猜拳“ 响应逻辑
   ├── index.css                页面样式
   ├── index.html               页面结构
   ├── index.js                 页面脚本
   └── server.js                http server
```
